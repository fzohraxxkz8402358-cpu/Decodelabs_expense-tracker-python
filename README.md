# 💰 Expense Tracker (Python)

A simple command-line Expense Tracker built with Python. This project allows users to enter expenses one by one, validates the input, calculates the total spending, and displays an expense summary with the average expense.

---

## 📌 Features

- Add multiple expenses
- Running total after each entry
- Calculates total expenses
- Calculates average expense
- Input validation
- Prevents negative values
- Handles invalid inputs gracefully
- Uses the accumulator pattern
- Beginner-friendly Python project
